# Terminal Cheatsheet

## Navigation
- `pwd` ā†’ print working directory
- `ls` ā†’ list files/folders
- `ls -la` ā†’ list all files including hidden files
- `cd foldername` ā†’ move into a folder
- `cd ..` ā†’ move up one folder

## File and Folder Creation
- `mkdir foldername` ā†’ create a folder
- `mkdir -p a/b/c` ā†’ create nested folders
- `touch file.txt` ā†’ create an empty file

## File Operations
- `cp source target` ā†’ copy file
- `mv old new` ā†’ move or rename file
- `rm file.txt` ā†’ remove a file
- `cat file.txt` ā†’ show file content

## Useful Concepts
- Relative path = based on current folder
- Absolute path = full path from root
